timetraveller

This is a simple web server that's intended to be used as test object when shifting kubernetes/openshift containers in time.

  • It shows the current time as text at /text/now and as seconds since 1970 at /text/epoch.
  • /text/uptime shows the uptime of the system/pod in seconds.
  • On /diff/yourTimeInSecondsSince1970 you'll find the difference between given and internal time.
  • Find more output under /time/xxx in different formats (e.g. /time/json); see Readme.md for details.
  • At /env/FAKETIME and /env/LD_PRELOAD it displays the content of the respective environment variable.
